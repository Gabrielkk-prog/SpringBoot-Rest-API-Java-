📚 Documentação da API REST

🎯 Sobre o Projeto

Esta é uma API REST desenvolvida com Spring Boot que se integra com o Supabase para recuperar dados.

🔗 Endpoints Disponíveis

GET /api/health - Verifica se o servidor está rodando

GET /api/dados?tabela=nome_tabela - Recupera dados de uma tabela do Supabase

🚀 Como Usar

  1. Abra o arquivo client.html no navegador
  2. Digite o nome da tabela do Supabase (ex: users, produtos)
  3. Clique em "Buscar Dados"

⚙️ Configuração

Edite o arquivo application.properties e configure:

  • supabase.url - URL do seu projeto Supabase
  • supabase.anon-key - Chave anônima do Supabase
